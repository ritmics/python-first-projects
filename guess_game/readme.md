# Финансовый Трекер 
Консольное приложение для учёта личных финансов на Python.

## Функционал
- Добавление доходов и расходов с категориями
- Автоматический расчёт текущего баланса
- Полная история операций с реальными датами
- Статистика по категориям (итоги)
- Проверка ввода (только числа, положительные суммы)

## Как запустить
1. Убедись, что установлен Python 3.x
2. Скачай файл `finance_tracker.py`
3. Запусти в терминале:
```bash
python finance_tracker.py
