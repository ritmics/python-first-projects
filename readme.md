# Guess the Number 

Простая консольная игра на Python. Компьютер загадывает число от 1 до 10, игрок угадывает.

## Особенности
- Подсказки "Больше"/"Меньше"
- Подсчёт попыток
- Проверка ввода (только числа)
- Чистый код с функцией

## Как запустить
1. Убедись, что установлен Python 3.x
2. Скачай файл `guess_game.py`
3. Запусти в терминале:
```bash
python guess_game.py
