# Конвертер единиц

Простая консольная программа для конвертации единиц измерения на Python.

## Что умеет
- Конвертировать температуру (Цельсий ↔ Фаренгейт)
- Конвертировать длину (Метры ↔ Километры)
- Конвертировать вес (Килограммы ↔ Граммы)
- Сохранять историю конвертаций

## Как запустить
1. Установи Python 3
2. Скачай файл converter.py
3. Запусти в терминале:
```bash
python converter.py
